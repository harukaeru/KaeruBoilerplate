# KaeruBoilerplate
Boilerplate files for myself

## Usage
```sh
$ ./boiler html5
<html>
    <head>
        <style>
        </style>
    </head>
    <body>
    </body>
</html>
```
```sh
$ ./boiler django/test
from django.test import TestCase


class FooTestCase(TestCase):
    def setUp(self):
        pass

    def test_(self):
        self.fail()
```
